What's Inside

Everything We Built

⚛️

Next.js & React Frontend

A TypeScript-based UI styled with hand-written Glassmorphism CSS — frosted-glass panels, gradients, and micro-animations.

🛰️

Serverless Backend

Next.js API routes securely relay user input to the AI provider — no API keys ever reach the browser.

🤗

Hugging Face Inference

Connected via the official @huggingface/inference SDK, auto-routed through router.huggingface.co to free-tier providers.

🧠

Qwen2.5-72B-Instruct

A massive open-source reasoning model acting as the "medical expert" core — far more capable than typical 7B models.

🌍

Localization Engine

Strict prompt engineering forces the AI to localize: real specialist types, city-relevant guidance, and prices in local currency.

💊

Medicine Cost Estimator

Estimates the cost of over-the-counter relief medicines based on the user's location and reported symptoms.

🗂️

Dynamic Tabbed UI

Parses the AI's structured JSON response and renders it live into a clean, organized, tabbed results interface.

🛡️

Resilient JSON Rendering

A recursive renderContent function safely converts deeply nested AI output into readable "Diagnostic Cards" instead of crashing.

⚡

Real Challenges Solved

DNS blackouts, model rejection errors, React rendering crashes, and terminal encoding bugs — all diagnosed and fixed.
