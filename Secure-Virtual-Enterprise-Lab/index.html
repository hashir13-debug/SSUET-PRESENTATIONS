What's Inside
Everything We Built
🖥️

VMware ESXi 8.0.3

Bare-metal hypervisor hosting all VMs. Installed as a nested VM inside Workstation Pro 26H1 on HP laptops.

🛡️

pfSense Firewall

Full gateway: WAN, LAN, DMZ, MGMT interfaces. NAT, DHCP, DNS, and strict inter-zone firewall rules.

🏢

Active Directory

Windows Server 2022 Domain Controller. Domain: cyberlab.local. OUs, users, groups, and Group Policies configured.

🐧

Linux Web Server (DMZ)

Ubuntu 22.04 with Nginx in DMZ zone. SSH hardened: key-only auth, root disabled, Fail2Ban active.

💻

Domain Clients

Windows 10/11 and RHEL 9 workstations joined to cyberlab.local. Authenticate via AD. GPOs apply on login.

🎩

RHEL 9 Domain Join

Red Hat Enterprise Linux 9 client joined to Active Directory (cyberlab.local) via SSSD — cross-platform identity management verified.

🐉

Kali Linux Testing

Kali Linux 2026.1 for penetration testing. Nmap scans validated all firewall rules — 100% confirmed.

🌐

VLAN Segmentation

Three isolated zones: LAN (10.x), DMZ (20.x), MGMT (30.x). DMZ→LAN blocked. Users→MGMT blocked.

📊

Monitoring & Logging

Windows Event Log, pfSense firewall logs, Linux auth.log, Nginx access log — full audit trail.

⚡

Real Challenges Solved

Hypervisor conflict (Hyper-V vs VMware), storage config, domain join DNS issue — all documented and solved.
